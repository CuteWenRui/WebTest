## JavaScript基础总结

> * JavaScript中区分大小写。
> * 每条语句结尾可以省略分号,但是最好加上,这样可以保证每行代码的准确性,提高可读性。
> * 变量是弱类型的,只能使用var运算符。
> * 注释代码有两种，单行注释和多行注释。
> * 空值：用于定义空的或者不存在的引用。
> * 未定义值：当使用了一个并未声明的变量，或者使用了一个已经声明但没有赋值的变量时，将返回未定义值(undefined)。
> * 如果只是声明了变量，但是未对其赋值，其默认值会是undefined。
> * js中还有一种特殊类型的数字变量NaN——非数字。当程序中由于某种原因发生计算错误后，将产生一个没有意义的数字，就是返回的NaN。
> * 变量的作用域：全局变量和局部变量。

等等，见js.html。
